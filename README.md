# RxSaver Vet

**Prescription Drug Cost Navigator for Seniors & Veterans**

Full-stack production-grade app ready for market. Leverages 2026 Medicare Part D changes: $2,100 OOP cap, negotiated prices on 10+ drugs, payment plans.

## Features
- Real-time price comparison (GoodRx + negotiated prices)
- 2026 OOP tracker & payment plan simulator
- VA/Medicare coordination
- Savings alerts & coupon finder
- Premium: $4.99/mo or affiliate commissions

## Tech Stack
Next.js 15, TypeScript, Tailwind, Stripe, Supabase

## Pricing Model
Free core + $4.99/mo Premium. B2B for plans/unions.

## GTM
VA hospitals, AARP, senior centers, Google ads for "high drug costs Medicare".

## Defensibility
Curated price DB + user savings proofs + integrations.
